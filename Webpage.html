Jason Wu's guide to create a website with HTML and CSS

Introduction:

The basics of HTML to create a website starts with an idea .
Any topic or interest of yours can be expressed through a website created through Hyper text markup language
to get started, you must have a program to write the code.
The following are programs that can be used:

  • Notepad
  • Notepad++
  • TextWrangler
  • Coda
  • Subline Text

Once you have a program started, you can start writing your code.
The first line for all websites must be: "< !DOCTYPE html >"
Code within HTML will have tags which are enclosed by "<" and ">" symbols.
Second, the next tag will be the head tag.


The Head:

The head tag will be the first tag you will encounter with a closing tag.
A closing tag is similar to a regular tag, but with a "/" infornt of the tag inside of the "<>".
For example: "< head >" would start the head of the website and "< /head >" would end it.

Within the head tags, there are many things that can be put into it.
  • "< meta >" which sets the website's character type.
  • "< title >" sets the title of the website, located on the tabs of the browser.
  • "< style >" to edit the design or the CSS of the website.

These tags will affect the overall webpage and not the specific content within the website.
The text and formatting of the text and images will be all done within the body tag.



The Body:

Like the head tag, the body tag will be "< body >" and will end in "< /body >".
Most of the content will be put within the body of the website from text to images.
The variety and options are limitless!
The following are only a few of the tags for the body:

  • "< p >" for paragraph text.
  • "< table >" for a table of organized text.
  • "< img >" for embedding images into the site.
  • "< h# >" where # consist of (1-5) for heading size for text.
